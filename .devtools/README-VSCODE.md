# VS Code Setup (OGZPrime)

## Import OGZPrime Profile (fastest)
1) Open VS Code
2) Drag & drop: `.devtools/OGZPrime.code-profile`
3) Preview + Import

## Open Workspace
Open: `.devtools/OGZPrime.code-workspace`

## Tasks (1-click Ops)
Open Command Palette:
- `Tasks: Run Task`
Then choose:
- OGZ: Compose Up
- OGZ: Compose Down
- OGZ: Logs (All)
- OGZ: Restart Bot / WebSocket / Dashboard / TRAI
